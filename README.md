# Sistema de Pedidos - Food Store

Aplicación en consola desarrollada en **Java 21** que permite gestionar categorías, productos, usuarios y pedidos.  
El proyecto aplica conceptos de **Programación Orientada a Objetos**, colecciones en memoria y manejo de excepciones personalizadas.

## Requisitos
- Tener instalado **Java JDK 21** o superior.
- Opcional: Tener instalado un IDE como IntelliJ, Eclipse o NetBeans.

## Compilacion y ejecución:
Si se usa la terminal:
- En la carpeta raiz del proyecto:

```bash
javac Main.java
java Main
```
Si se usa un IDE como NetBeans:

- Dentro del programa, ir a Importar Projecto -> From Zip... y se selecciona el archivo zip descargado.

- O se ingresa a la carpeta C:\Users\<Nombre>\Documents\NetBeansProjects y se deja la carpeta que esta dentro del archivo zip.
